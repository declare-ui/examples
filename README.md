# DeclareUI Examples

Example projects and starter templates demonstrating DeclareUI with each target framework.

---

## Examples

| Example | Description |
|:--------|:------------|
| `react-starter` | React + TypeScript + Tailwind |
| `vue-starter` | Vue 3 + Composition API + Tailwind |
| `angular-starter` | Angular standalone components + Tailwind |
| `svelte-starter` | SvelteKit + runes + Tailwind |
| `web-components-starter` | Custom Elements with Shadow DOM |
| `vanilla-starter` | Vanilla JS/TS, zero dependencies |
| `multi-target` | Single project compiling to all frameworks |

## Running an example

```bash
cd react-starter
pnpm install
pnpm dev
```

## Creating your own project

Use the scaffolding tool instead:

```bash
npx create-declareui my-project --targets react,vue
```
